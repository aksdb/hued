go-hue-web
==========
A tablet/phone web interface using Philips HUE built just for me.

setup / installation
---
	$ curl https://glide.sh/get | sh
	$ go get github.com/heatxsink/go-hue-web
	$ cd $GOPATH/src/github.com/heatxsink/go-hue-web/
	$ glide install
	$ go build -v hued.go
